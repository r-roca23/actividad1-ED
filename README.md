En esta actividad el alumno debe realizar de forma 
individual un trabajo práctico de control de versiones 
con GitHub y Visual Studio Code, documentando todo el 
proceso en un único informe PDF. 1. Primero, accede a 
GitHub, crea un repositorio nuevo con README inicial y 
visibilidad privada; captura pantalla del proceso y anota 
en el informe el nombre elegido y la configuración 
seleccionada. 2. Abre Visual Studio Code, y clona tu 
repositorio pegando la URL HTTPS; incluye en el informe 
capturas de tu explorador de archivos y de la ventana de 
VSC mostrando el proyecto abierto. 3. Desde la interfaz 
web de GitHub edita directamente el README.md: 
añade descripción del proyecto, tu nombre, la fecha y un 
resumen de lo hecho hasta el momento; guarda los 
cambios y muestra antes/después con capturas. 4. Crea 
un archivo nuevo desde GitHub (“Add file → Create new 
file”) llamado print.py con un sencillo 
print("Hola, mundo"), guarda y captura la pantalla 
del editor web. 5. Regresa a Visual Studio Code, abre 
print.py, añade otra línea de código (por ejemplo, 
print("Editado desde VSC")), guarda los cambios 
y prepara un mensaje de commit (“Editado desde VSC”). 
6. En la pestaña de control de versiones de VSC realiza el 
commit y ejecuta “Push” para sincronizar con el 
repositorio remoto; verifica en GitHub que los cambios 
aparecen y captura el historial de commits. 7. Durante 
todo el proceso, anota cualquier incidencia (errores de 
autenticación, conflictos, extensiones que faltan) y 
describe en el informe cómo las resolviste. 8. 
Finalmente, redacta una reflexión de al menos tres 
ventajas (por ejemplo, historial de cambios, 
colaboración sencilla, integración IDE) y tres 
inconvenientes (por ejemplo, complejidad inicial, 
posibles conflictos, dependencia de internet) del uso de 
la extensión GitHub en Visual Studio Code. El informe 
debe titularse Act1_2ev_Nombre_Apellido.pdf, incluir el 
enlace al repositorio, todas las capturas mencionadas y 
enviarse al aula virtual antes del 12 de enero de 2025# actividad1-ED
